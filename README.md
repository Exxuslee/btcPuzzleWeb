**Скрипт поиска пазла биткоина №64**

1 поток выбирает случайным образом старшие 10 символов и полным перебором проверяет оставшиеся 6 символов.

## Starting
To make http:
```
npm run start
```
To start server, just run:
```
npm run rserver
```
